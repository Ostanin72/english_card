# EnglishCard — Документация проекта

## EnglishCard — это веб-приложение для изучения английского языка, разработанное с использованием фреймворка Streamlit. Приложение позволяет пользователям проходить интерактивные тесты, пополнять свой личный словарь и отслеживать статистику прогресса.

### 1. Обзор архитектуры
Проект имеет модульную структуру, разделенную на три основных логических блока:

main.py: Точка входа приложения. Отвечает за общую настройку страницы, управление состоянием сессии (авторизация) и отрисовку основного контента в зависимости от статуса пользователя.
ui/sidebar.py: Модуль пользовательского интерфейса. Содержит функции для отрисовки боковой панели (авторизация) и всех основных вкладок приложения: «Изучение», «Добавить слово», «Удалить слово», «Статистика».
db/init_db.py: Модуль инициализации базы данных.
db/user.py: Модуль взаимодействия с базой данных. Реализует всю бизнес-логику по работе с пользователями, словами и статистикой через SQL-запросы.

Приложение предоставляет следующий функционал:

Пользователь входит в систему под своим именем. Если пользователь не существует, система автоматически создает его.

### Основной режим обучения. 
Пользователю предлагается русское слово и четыре варианта ответа на английском языке. Система отслеживает правильность ответов.

### Персональный словарь:
### Добавление слов: 
Пользователь может добавить в свой словарь слова, которых нет в общей базе.

### 3. Инструкция по установке и запуску
Для запуска приложения на локальной машине необходимо выполнить следующие шаги:

Установить Python (>=3.8), если он не установлен.
Создать виртуальное окружение (рекомендуется):

python -m venv venv

Активировать виртуальное окружение:
На Windows: venv\Scripts\activate
На Linux/macOS: source venv/bin/activate

Установить зависимости из файла requirements.txt:

pip install -r requirements.txt

Настроить базу данных:
Убедитесь, что у вас установлена СУБД PostgreSQL.
Создайте базу данных и пользователя.
Настройте параметры подключения в файле .env (.env.example - образец).

Запустить приложение:

streamlit run main.py

Открыть приложение: Запустится браузер по умолчанию с интерфейсом EnglishCard.

### 4. Описание работы с базой данных
Приложение использует реляционную базу данных PostgreSQL со следующей структурой таблиц:

users: Хранит информацию о пользователях (ID, имя).
common_words: Содержит общий словарь слов (ID, русское слово, английский перевод).
user_words: Хранит персональные слова пользователей (ID, ID пользователя, русское слово, английский перевод).
learning_stats: Ведет журнал попыток изучения слов (ID, ID пользователя, ID слова, тип слова, результат, дата).

Логика ключевых операций:

Вход пользователя (login_user):
Выполняется поиск пользователя по имени.
Если пользователь найден, возвращается его ID.
Если не найден — создается новая запись в таблице users, и возвращается ID нового пользователя.

Получение слов для изучения (get_user_words):
Формируется единый список слов путем объединения (UNION ALL) данных из таблиц common_words и user_words конкретного пользователя.

Добавление своих слов для обучения (get_personal_words):
Добавляет пары слов (слово-перевод) в пользовательский словарь.

Удаление своих слов (delete_personal_word):
Удаляет ранее добавленные пары слов (слово-перевод) из пользовательского словаря.

Обновление статистики (update_stats):
При каждой попытке ответа (как верной, так и неверной) в таблицу learning_stats добавляется новая запись.
Расчет агрегированной статистики (get_statistics):

Для получения итоговых данных используются сложные SQL-запросы с CTE (Common Table Expressions), которые позволяют за один запрос вычислить общее число попыток, число правильных ответов и количество уникальных выученных слов.
